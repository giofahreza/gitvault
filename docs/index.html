Sovereign sync

Your GitHub repository becomes encrypted storage.

GitVault encrypts the vault before anything leaves the device. GitHub stores opaque files only, while the keys and recovery phrase stay with you. 

gitvault vault seal
  status: encrypted
  cipher: XChaCha20-Poly1305
  target: private GitHub repo

gitvault sync push
  uploaded: vault_index.enc
  uploaded: devices.enc
  readable secrets: 0
