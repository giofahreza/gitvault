Setup

Use your normal GitHub account as encrypted storage.

  1. Create an empty private repository in your GitHub account.
  2. Create a fine-grained GitHub token with metadata read and contents read/write access.
  3. Open GitVault, save your recovery kit, then connect GitHub Sync in Settings.
  4. Install the Android APK when you need autofill, keyboard, biometric, and mobile background sync.

Security model

Your vault is encrypted on device.

GitVault uses XChaCha20-Poly1305 authenticated encryption. GitHub only stores encrypted vault files, while your recovery phrase, PIN, and readable secrets stay on your devices.
